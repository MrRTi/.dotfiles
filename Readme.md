# Dotfiles

## Dependencies
- [Stow](https://www.gnu.org/software/stow/)
    -   `brew intall stow` (should be included in Brewfile)

## How to set configs

1. Clone repo
2. `cd` into repo folder
3. Run `./stow.sh`
